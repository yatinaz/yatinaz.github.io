Multi-Disease fMRI Biomarker Detection

Deep learning framework for schizophrenia and Parkinson's disease detection from resting-state fMRI functional connectivity matrices. Deployed as a FastAPI service in Docker. University of Western Australia thesis project.

Schizophrenia AUC 0.718 542 subjects FastAPI · Docker · nilearn

Schizophrenia — Figures

Schizophrenia ROC curves. AUC 0.718 on 542 subjects across multiple sites. The multi-site nature makes this harder than single-site benchmarks — site effects add confounding variance.

Functional connectivity heatmap. Average FC matrix across schizophrenia vs. control subjects. Look for the visual difference in connectivity between the default mode network and frontal regions.

Glass brain projection. nilearn glass brain showing the strongest discriminative connectivity edges. Connections involving the prefrontal cortex and cingulate are most predictive.

Feature importance. Top FC edges ranked by model contribution.

Confusion matrix. True positive rate vs. false alarm rate on held-out subjects.

SHAP summary. SHAP values showing which FC connections push predictions toward schizophrenia (red) vs. control (blue). Each dot is one subject.

Per-site comparison. Performance broken down by scanner site. Site-to-site variation is the main barrier to clinical deployment — harmonisation is critical.

Development history. Iterative model improvement from baseline logistic regression through the final multi-modal pipeline.
